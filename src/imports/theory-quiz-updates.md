Нужно внести правки только в страницу Theory (аккордеоны).  
Ничего не менять в layout, размерах карточек, контейнерах и стилях.  
Исправляем только текст и поведение ответов.

ПРАВИЛА:

1. Не изменять структуру страницы.
2. Не менять размеры карточек.
3. Не добавлять новые контейнеры.
4. Не менять стили layout.
5. Работать только с текстом и логикой ответа внутри мини-квиза.

---

ИЗМЕНЕНИЯ В МИНИ-КВИЗЕ:

Если пользователь выбирает правильный ответ:
- показывается зелёный блок ответа
- рядом с текстом отображается иконка галочки
- текст ответа отображается под карточками
- появляется кнопка «Продолжить»
- в аккордеоне слева чёрная точка меняется на зелёную иконку

Если пользователь выбирает неправильный ответ:
- появляется красный блок ответа
- в начале блока должна быть иконка кулака
- не добавлять вторую иконку
- текст должен быть коротким

Пример текста:

Иконка кулака  
Почти получилось. Попробуй другой вариант.

После этого пользователь может выбрать другую карточку.

ВАЖНО:
Неправильный ответ не блокирует вопрос.  
Пользователь должен иметь возможность выбрать другой вариант и ответить правильно.

СТРУКТУРА ТЕКСТА В АККОРДЕОНЕ:

Текст теории должен быть одинаково оформлен:

• использовать иконки для смысловых блоков  
• использовать буллиты  
• выделять важные слова курсивом

🎯 Заголовок блока

• цвет

• расстояние

Если элементы слишком похожи, пользователь теряется.
