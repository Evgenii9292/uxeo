Нужно внести несколько исправлений.

ВАЖНО:

Не менять layout, структуру страницы, scroll или стили.

Исправить только логику ниже.

---

1. СОХРАНЕНИЕ НЕПРАВИЛЬНОГО ОТВЕТА

Если пользователь отвечает неправильно в мини-квизе:

• появляется оранжевый крестик  
• показывается объяснение  

Но если пользователь выходит из теории и возвращается обратно:

неправильный ответ сбрасывается.

Аккордеон снова должен показывать:

черную точку.

Это сделано специально, чтобы пользователь мог пройти урок заново.

---

2. КНОПКА ПОСЛЕДНЕГО АККОРДЕОНА

В последнем аккордеоне должна быть кнопка:

Начать квиз

Правила:

• слева должна быть оранжевая иконка молнии  
• текст кнопки: Начать квиз

ВАЖНО:

Кнопка должна отображаться всегда.

Даже если пользователь ответил неправильно в мини-квизе.

Сейчас кнопка появляется только если ответы правильные — это нужно исправить.

---

3. КАРТОЧКИ МИНИ-КВИЗОВ

Карточки должны содержать:

• изображение интерфейса  
• изображение должно быть по центру

Не использовать текст:

Интерфейс A  
Интерфейс B

Карточки должны использовать универсальные компоненты:

QuizCardLeft  
QuizCardRight

---

4. ВАЖНО

Не изменять:

• layout  
• scroll  
• размеры карточек  
• структуру страницы
