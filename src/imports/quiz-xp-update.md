Update quiz XP behavior for replayed lessons.

XP rules:

XP is awarded ONLY the first time a lesson is completed.

Example:

1 correct answer = 25 XP

If the lesson is replayed:

• XP must NOT be awarded  
• XP indicators should NOT appear

First lesson completion:

Correct answer:

"Верно!"
+25 XP

Wrong answer:

"Почти получилось"

Replay mode (lesson already completed):

Correct answer:

"Верно!"

Wrong answer:

"Попробуй ещё"

Important:

Do NOT show "+0 XP".  
Instead hide the XP reward entirely in replay mode.

If a lesson is already completed:

• Button text = "Повторить"
