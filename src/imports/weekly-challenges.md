Задача: воспроизвести экран "Недельные вызовы" по прикреплённому макету.

Сначала внимательно изучи прикрепленный макет и структуру слоев.
После этого воспроизведи интерфейс максимально точно.

Это НЕ задача на генерацию нового дизайна.
Это задача на точное воспроизведение layout.

--------------------------------------------------

IMPORTANT RULES

1. Работай только внутри выбранного frame.

2. Не создавай новый layout.

3. Не генерируй новый дизайн.

4. Используй существующие компоненты и стили из файла.

5. Не создавай новые text styles.

6. Не создавай новые кнопки.

7. Не изменяй:

sidebar  
header  
правую колонку widgets  
grid  
spacing системы

8. Меняется только центральная часть страницы.

9. Layout должен максимально повторять прикрепленный макет.
