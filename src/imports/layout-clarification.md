Important layout clarification.

The application uses TWO layout types.

Do not mix them.

---

LAYOUT TYPE A

Used for:

Courses page  
Challenges page  
Profile page  

Structure:

Sidebar  
Header  
MainContent  
RightWidgets

RightWidgets include:

WeeklyChallenge  
LeagueProgress  
DailyTask

These widgets appear on the RIGHT SIDE.

---

LAYOUT TYPE B

Used for:

Course page (lessons screen)

Structure:

Sidebar  
Header  
MainContent  
Roadmap (RIGHT SIDE)

Widgets are NOT on the right here.

Instead they appear BELOW the main card.

Example order:

CourseCard  
WeeklyChallenge  
LeagueProgress  
DailyTask

Roadmap stays on the RIGHT SIDE.

---

IMPORTANT

Do not move Roadmap.
Do not replace Roadmap with widgets.

Widgets only move below the card on Course page.
